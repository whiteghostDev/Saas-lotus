Creates a credit.

<CodeGroup>

```py Python
lotus.create_credit(
    customer_id='cust_0569173ee6654369',
    amount=120.00,
    currency_code='USD',
)

lotus.create_credit(
    customer_id='cust_0569173ee6654369',
    amount=1000.00,
    currency_code='USD',
    description="Prepurchased credit",
    expires_at="2024-01-01T00:00:00Z",
    amount_paid=500.00,
    amount_paid_currency_code='USD',
)
```

```ts Typescript
await lotus.createCredit({
  customerId: "cust_0569173ee6654369",
  amount: 120.00,
  currencyCode: "USD"
});

await lotus.createCredit({
  customerId: "cust_0569173ee6654369",
  amount: 1000.00,
  currencyCode: "USD",
  description: "Prepurchased credit",
  expiresAt: "2024-01-01T00:00:00Z",
  amountPaid: 500.00,
  amountPaidCurrencyCode: "USD"
});
```

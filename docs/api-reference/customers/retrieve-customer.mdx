---
title: "Retrieve a customer"
openapi: "GET /api/customers/{customer_id}/"
hideApiMarker: true
---

Retrieves a [customer_detail](/api-reference/customers/customer-object) object.

A Retrieve `Customer` call requires

- `customer_id` the id you defined in your backend for the corresponding customer and the same id that you passed into Lotus when creating the customer

<CodeGroup>
```py Python
lotus.get_customer(
  customer_id='cust_0569173ee6654369',
)
```

```ts Typescript
await lotus.getCustomerDetail({
  customerId: "cust_0569173ee6654369",
});
```

</CodeGroup>

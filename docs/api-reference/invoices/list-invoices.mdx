---
title: "List invoices"
openapi: "GET /api/invoices/"
---

Retrieves an array of invoice objects.

<CodeGroup>

```py Python
lotus.list_invoices(
    customer_id="cus_1234567890",
)

lotus.list_invoices(
    status=["unpaid"],
)
```

```ts Typescript
await lotus.listInvoices({
    customerId="cus_1234567890",
});

await lotus.listInvoices({
    status=["unpaid"],
});
```

</CodeGroup>

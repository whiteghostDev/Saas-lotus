openapi: "GET /api/plans/{plan_id}/"
title: Get Plan
---

Retrieves a [plan_detail](/api-reference/plans/plan-object) object.

A `Get Plan` call requires

- `plan_id` the plan id

<CodeGroup>

```py Python
lotus.get_plan(
    plan_id='plan_0569173ee6654369',
)
```

```ts Typescript
await lotus.getPlan({
  planId: "plan_0569173ee6654369",
}); // Will return a promise
```

</CodeGroup>
